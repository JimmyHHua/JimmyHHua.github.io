二、时间安排

每周具体学习时间划分为4个部分:

  1. 部分安排周一到周二
  2. 部分安排在周四到周五
  3. 部分安排在周日
  4. 部分作业是是任何有空的时间自行完成，可以落后于学习进度
  5. 周三、周六休息

四、学习计划

Week 1:

  • 第1部分学习任务：
    （1）听深度学习绪论+0基础一小时完成一场比赛PPT
    学习时长：10/30
    任务详解：视频1主要是介绍深度学习的一些应用案例，第二个是完成一次kaggle比赛的流程
    （2）了解计算机视觉综述，历史背景和课程大纲
    学习时长：10/30
    slides: lecture01（链接： https://github.com/sharedeeply/cs231n-camp/blob/master/slides/cs231n_2018_lecture01.pdf ）
    观看视频 p1, p2 和 p3
    课程视频： https://www.bilibili.com/video/av17204303 /

  • 第2部分学习任务：
    （1）学习数据驱动的方法, 理解 KNN 算法，初步学习线性分类器
    slides: lecture02（ https://github.com/sharedeeply/cs231n-camp/blob/master/slides/cs231n_2018_lecture01.pdf ）
    观看视频 p4 p5 和 p6（ https://www.bilibili.com/video/av17204303 /）
    学习 图像分类笔记上（ https://zhuanlan.zhihu.com/p/20894041?refer=intelligentunit ）
    学习 图像分类笔记下（ https://zhuanlan.zhihu.com/p/20900216 ）
    学习 线性分类笔记上（ https://zhuanlan.zhihu.com/p/20918580?refer=intelligentunit ）
    学习时长：11/1—11/2

  • 第3部分学习任务：
    （1）掌握本门课 python 编程的基本功
    阅读 python 和 numpy 教程（ https://zhuanlan.zhihu.com/p/20878530?refer=intelligentunit ）和代码（ https://zhuanlan.zhihu.com/p/20878530?refer=intelligentunit ）
    学习时长：11/4

  • 第4部分作业：(热身)写一个矩阵的类，实现矩阵乘法，只能使用 python 的类(class)和列表(list)
    完成assignment1 中的 knn.ipynb
    作业链接： https://github.com/sharedeeply/cs231n-camp/tree/master/assignment/assignment1

  • 第1部分学习任务：

    学习 线性分类笔记下（ https://zhuanlan.zhihu.com/p/21102293 ）
    学习时长：11/5—11/6

  • 第2部分学习任务：
    （1）学习损失函数以及梯度下降的相关知识
    slides: lecture03（ https://github.com/sharedeeply/cs231n-camp/blob/master/slides/cs231n_2018_lecture03.pdf ）
    观看视频 p8（ https://www.bilibili.com/video/av17204303 /）
    学习 最优化笔记（ https://zhuanlan.zhihu.com/p/21360434?refer=intelligentunit ）
    学习时长：11/8—11/9

  • 第3部分学习任务：
    （1）掌握矩阵求导的基本方法

    (1)简述 KNN 和线性分类器的优劣

    （1）学习掌握深度学习的基石: 反向传播算法

  • 第2部分学习任务：

  • 第3部分学习任务：
    （1）深入理解反向传播算法
    阅读反向传播算法的数学补充（ http://cs231n.stanford.edu/handouts/derivatives.pdf ）和例子（ http://cs231n.stanford.edu/handouts/linear-backprop.pdf ）

  • 第4部分作业：

Week 4:

  • 第1部分学习任务：
    （1）掌握 PyTorch 中的基本操作
    学习 pytorch 的入门基础（ https://pytorch.org/tutorials/beginner/deep_learning_60min_blitz.html ）
    学习时长：11/19—11/20

  • 第2部分学习任务：
    （1）了解 kaggle 比赛的流程，并完成第一次的成绩提交
    了解比赛房价预测（ https://www.kaggle.com/c/house-prices-advanced-regression-techniques ）
